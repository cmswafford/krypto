Features

  • Human-input solutions are checked using the Wolfram API.
  • Feedback box displays log data.
  • Solver tests 256 combinations, trying all combinations of equations using all algebraic operators (+, -, *, /) in all possible locations between the numbers that are given in a random order.
  • Display subpages in friendly facebox design.
  • Simulator allows user to try and solve many games and see the results.
